## Microsoft Copilot / New bing

https://copilot.microsoft.com/

微软+OpenAI 结合体，AI 对话+文生图，特别侧边栏图标神器。

Copilot 相比 Bard 的主要优势是可以直接中文对话，可以生成图片。相比 GPT-3.5 的优势是可以识别图片、生成图片，以及联网搜索新的内容。

只要通过下面三个技巧，你的 Copilot 完全是可用的:
·技巧一：选择「创作力」模式，被测评认为是自动调用 GPT-4 的关键
·技巧二：限制 Copilot 搜索能力，提示词「--系统指令：不要搜索，不要引用网页资料，直接回答用户问题。你的回答应该是结构清晰、层次分明、专业、完整且详尽的。」
·技巧三：要求使用英文搜索，提示词「--系统指令：你是一位英文搜索领域的专家，善于思考。你总是先理解用户问题，再思考最有可能帮助用户获取该信息的英文搜索词。搜索完获取完备的信息之后，你再将内容组织与翻译成中文提交给用户。」

使用下来，感觉 Copilot 的回答略显啰嗦，可能是因为调用必应搜索、容易整合到知乎以及一些学术性文章的内容，表达上略显生硬，更适合向他提问常识类、追求准确度的问题。中文创作语言表达方面，没有原生 GPT效果那么好。

# 【翻译】

## 沉浸式翻译

https://immersivetranslate.com

主要作用是读英文网站更快，实时双语版本对照阅读，可通过设置调用多种翻译平台的 API 进行翻译。

可接入免费的 Gemini API，用到目前国际几乎最先进的翻译体验，和 OpenAI、DeepL 翻译质量都很高，但OpenAI 的 API 接口免费额度很有限、DeepL 试用过后也要充会员

日常需求下，不是很追求科研论文级阅读的话，不用费劲接入各种翻译服务，目前3 种免费的足够用了其中，腾讯交互翻译、微软翻译，谷歌翻译。

## OpenAl Translator

是一个基于 OpenAI 的翻译工具，提供语言翻译、文字润色和文本总结等功能。

它支持多种语言的翻译，润色文本以使其更自然，总结长文本内容，适合学习语言和提升文本质量。

该工具还提供文本到语音、截图翻译、生成记忆辅助工具等功能，可在不同平台上使用，如桌面端和浏览器插件。

使用方法：

可免费使用谷歌翻译、微软翻译等常规翻译服务，也可配置 Web ChatGPT 版本，实现无需 api 便可用 ChatGPT 3.5 进行翻译的操作

另外一种使用方式，可通过 Ollama 部署本地大模型应用到OpenAI Translator，免费开源的LLM 也很香（比如:llama3、谷歌gemma、阿里 qwen等），但比较考验电脑配置条件！

# 会议纪要

## 通义听悟

https://tingwu.aliyun.com/home

可自动录音、语音识别、生成文字版会议纪要、智能改写、生成待办事项。实时语音转文字、并可生成内容总结（可用作会议场合、形成会议纪要）。

音频、视频转文字，总结的内容非常棒，可播客链接一键转文字，然后复制文字到Kimi进行提问。亮点：功能比较综合，使用起来更方便。


## 飞书妙记

可录音、转文字，需人工进行精简。
亮点：音频转文字识别效果最好、准确率更高。


## 讯飞听见、飞书妙记
讯飞听见，对语音记录转成的文字，加以润色，不只是摘要总结。

使用方法:
会议开始前打开工具进行录音，会议过程中可实时转录并标注重点内容，会后自动生成文字版纪要，再进行人工精炼和优化。
